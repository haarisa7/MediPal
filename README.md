# Forecaster

Forecaster is a stock market prediction tool that uses a combination of stock market and weather attributes to predict movements for a given stock or index. This project is part of the software engineering project for university

## Installation
clone the repo 
```bash
git clone https://github.com/19Amathew/forecaster.git
```

## Usage
We are running the streamlit app in a docker container run below makefile command, check image configuration in Dockerfile
```python
make run-container 
```
Alternatively you can run locally
```python
make run
```
